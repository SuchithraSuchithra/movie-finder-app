# Movie Finder App

Movie Finder app helps to find the movie related information. All the information is fetched from the TMDB api https://developer.themoviedb.org
It has the following functionalities

### `Browse Movies`

### `Browse Cast/Crew`

### `Search Movies`

### `Movie Details`

### `Cast/Crew Details`

### `Upcoming Movies Details`
